Для игры в dungeon and dragons необходим мастер (Дм/Гм), который будет вести повествование и вести персонажей игроков по своей сюжетной линии. Игроки должный создать своего персонажа, к счастью данный сайт предоставляет набор функций для лёгкого создания персонажа, понятный для новичка и крайне удобный для продвинутых игроков.
